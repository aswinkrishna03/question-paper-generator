AI-Powered Question Paper Generator

Automatically generate exam-ready question papers from syllabus PDFs using advanced Natural Language Processing and Machine Learning.

About the Platform

The AI Question Paper Generator is designed to help educators, institutions, and students create structured examination papers instantly. By analyzing syllabus content, extracting key topics, and classifying them intelligently, our system generates balanced and relevant questions automatically.

Core Features

Smart Topic Extraction

Extracts meaningful academic concepts from structured PDF syllabus documents.

AI Question Generation

Uses transformer-based NLP models to generate definition and descriptive questions.

Automatic Mark Allocation

Classifies questions and assigns appropriate marks automatically.

Instant PDF Export

Generates a formatted downloadable question paper in PDF format.

How It Works

  1. Upload a text-based syllabus PDF.
  2. The system extracts and filters academic topics.
  3. AI model generates structured questions.
  4. Questions are compiled into a formatted question paper.
  5. Download your ready-to-use PDF instantly.

Technology Stack

Built using Flask (Python), NLP Transformers (T5 Model), PDF processing libraries, and structured database handling. The system combines backend intelligence with a clean, responsive frontend interface.
